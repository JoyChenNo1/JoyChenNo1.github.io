Docker 基础镜像构建示例

发表于2024-01-14|更新于2024-01-15|Docker

背景

服务更新基础镜像.

镜像Dockfile

FROM python:3.7

WORKDIR /usr/src/app
COPY ./watermark ./
COPY ./requirements.txt ./

RUN apt-get update && apt-get install -y vim libgl1-mesa-glx &&  pip install -r requirements.txt  -i https://pypi.tuna.tsinghua.edu.cn/simple && pip uninstall -y PyMuPDF && pip install PyMuPDF -i https://pypi.tuna.tsinghua.edu.cn/simple && pip uninstall -y opencv-python opencv-contrib-python opencv-python-headless && pip install opencv-python==4.7.0.72

WORKDIR /root

COPY ./start_server.sh /root/

构建命令

1、docker build

docker build  --platform=linux/amd64 -t watermark:2.0 --no-cache .

2、docker run

docker --rm watermrak:2.0 bash

docker run -d -t --name watermark_server_2_0 watermrak:2.0 bash
