工作中常用的命令行汇总

发表于2022-09-13|更新于2024-04-02|jobLinux

前言

日常使用命令积累，持续更新中。


Linux 命令

tar

(1) 压缩文件

# 压缩文件夹指定路径和输出路径
tar -zcvf dest.tar.gz /src_dir

(2) 解压文件

tar -zxvf

gzip


unzip

unzip -d dest_dir src.zip

scp

scp -r root@10.10.27.103:/root/src_files ./dest_file

docker

(1) save 镜像

docker save xxx_server:1.0 | gzip > path/to/save/xxx_server_1.0.tgz

(2) run 镜像

docker run -rm -it --name mynginx -d nginx:1.0 bash # 临时run镜像

docker run -d -t --name watermark_server watermark:1.1  bash

(3) 加载镜像

docker load < watermark_2.0.tgz

(4) 基于现有容器更新基础镜像

docker commit fb8fe watermark:2.0

(5) build 镜像

docker build -t docker.dev.datacloak.com/datacloak/switch-init:${tag} .

(6) harbor login

docker login docker.dev.datacloak.com

(7) push 镜像

docker push docker.dev.datacloak.com/datacloak/switch-init:${tag}

tcpdump


ab


netstat


ps


free


date

(1) 时间戳获取及转换

# 时间->时间戳： 
date +%s
# 时间戳->时间： 
date +%Y:%m:%d -d @1425384141

(2) 设置时间(慎用!!!)


grep

(1) 不区分大小写，多个关键字过滤

tail -f config_server.log | grep -iE "keyword1|keyword2" -C 4

(2) 反向过滤

grep -v "log_sdk" # 反向过滤

pgrep



Windows 命令

ipconfig

# 刷新 dns
ipconfig /flushdns

find

查找字符串


netstat

查看网络端口状态


move

移动文件


cls

清除屏幕
